# Beginner Programs 🛠️

This section gives you small projects to practice what you've learned so far.

## In this section

You'll build things like:
- a calculator
- unit converters
- an interest calculator
- a countdown timer

## Goal

By the end of this section, you'll start feeling more comfortable turning ideas into code.

## Tip

Projects are where learning starts to feel real. Don't worry if your code isn't perfect.
