# Games and Console Projects 🎮

This section is all about building fun projects in the terminal.

## In this section

You'll build things like:
- number guessing games
- rock paper scissors
- dice rollers
- a banking program
- a slot machine
- hangman

## Goal

By the end of this section, you'll have more confidence writing longer programs on your own.

## Tip

These projects combine lots of beginner topics together, which makes them great practice.
