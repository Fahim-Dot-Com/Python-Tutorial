# Object-Oriented Programming 🧱

Object-oriented programming, or OOP, is a way of organizing code using classes and objects.

## In this section

You'll learn:
- classes and objects
- class variables
- inheritance
- polymorphism
- abstract classes
- composition
- aggregation
- decorators
- properties
- magic methods
- data classes

## Goal

By the end of this section, you'll understand how bigger Python programs can be structured more neatly.

## Tip

OOP can feel strange at first. Take it slowly and focus on one concept at a time.
