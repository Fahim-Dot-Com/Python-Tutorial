# Strings 📝

Strings are how Python works with text.

## In this section

You'll learn:
- string methods
- indexing
- slicing
- formatting text
- text-based mini projects

## Goal

By the end of this section, you'll be able to read, change, and format text in useful ways.

## Tip

Text manipulation is one of the most common things you'll do in programming, so this section is worth practicing a lot.
