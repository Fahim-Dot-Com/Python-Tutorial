# Functions 📞

Functions let you reuse code and keep programs organized.

## In this section

You'll learn:
- how to create functions
- parameters and return values
- default arguments
- keyword arguments
- `*args` and `**kwargs`
- scope
- lambda functions
- recursion

## Goal

By the end of this section, you'll be able to break big problems into smaller reusable pieces.

## Tip

If you copy and paste the same code more than once, that might be a sign you should use a function.
