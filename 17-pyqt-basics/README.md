# PyQt Basics 🖥️

This section introduces GUI programming with PyQt5.

GUI stands for Graphical User Interface — apps with windows, buttons, labels, and input boxes.

## In this section

You'll learn:
- how to set up a PyQt app
- labels
- images
- layout managers
- buttons
- checkboxes
- radio buttons
- line edits
- styling with CSS

## Goal

By the end of this section, you'll know how to build simple desktop app interfaces in Python.

## Tip

GUI programming is different from console programming, so take your time with it.
