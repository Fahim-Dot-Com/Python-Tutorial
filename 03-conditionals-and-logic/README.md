# Conditionals and Logic 🤔

Programs become much more useful when they can make decisions.

## In this section

You'll learn:
- `if`
- `elif`
- `else`
- logical operators
- conditional expressions
- membership operators
- `match-case`

## Goal

By the end of this section, you'll be able to make your code respond differently depending on conditions.

## Tip

Read conditions like questions:
- Is this true?
- If yes, do this.
- If not, do something else.
